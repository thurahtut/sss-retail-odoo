INVENTORY ADJUSTMENT COST PRICE

This module gives You opportunity to make an inventory adjustment which includes product cost. This product cost is saved in exact stock move for that product. If You are using FIFO method, system will use the product by it's incoming date and also the cost.

Let's take a better look:

In the inventory adjustment form You see new field "Unit Cost". This cost price is written in for each unit. For example: Product "Unit Cost" is 3.99 €, Real Quantity set to 5 units. Each product will have a cost price of 3.99 € and overall stock move cost will be 19.95 €.

Inventory valuation

Inventory valuation report shows, that adjustment value we've made is 19.95 €. If we create a new inventory adjustment with new cost price, we'll see new line of new stock move and also additional line in the inventory valuation report. In the picture shown, You can clearly see that additional adjustment for product "Product" value is 24.95 €. We created new inventory adjustment which added 5 more units and "Unit Cost" were 4.99 €.

NOTE!

If you leave "Unit Cost" field empty, with 0.00 inside. Odoo standard functionality will be used instead.
